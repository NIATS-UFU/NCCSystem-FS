Instructions to create a deployable .app (for QT application on Mac OS X)

1 - After build the application, usually an .app will be created.
2 - Type into the terminal: $ QTDIR/bin/macdeployqt <path to app file generated by build>/appFile.app
3 - Done! Your .app file size will be bigger.
4 - You can now distribute this .app to be used.

Ref.:
- https://doc-snapshots.qt.io/qt5-5.11/osx-deployment.html
- https://riptutorial.com/qt/example/24344/deploying-on-mac
